# Objetivo do Trabalho

Desenvolver uma aplicação prática utilizando **Lógica Fuzzy** para resolver um problema real ou simulado, implementando a solução em **Python no Google Colab**.

O objetivo é aplicar os conceitos estudados em sala, incluindo:

- variáveis linguísticas,
- funções de pertinência,
- regras fuzzy,
- inferência,
- defuzzificação,
- interpretação dos resultados.

***

# Requisitos do Trabalho

O grupo deverá:

## 1. Escolher um problema

O problema deve ser simples, porém coerente com o uso de Lógica Fuzzy.

Exemplos:

- controle de velocidade,
- temperatura de ambiente,
- nível de risco,
- consumo de energia,
- irrigação automática,
- luminosidade,
- qualidade de serviço,
- nível de trânsito,
- recomendação simples,
- entre outros.

> **⚠️ Não serão aceitas cópias/plágios de aplicações prontas encontradas na internet.**

A modelagem deve ser autoral e adaptada pelo grupo.

## 2. Explicar o problema

No próprio notebook Colab, apresentar:

- descrição do problema,
- objetivo da solução,
- justificativa do uso da Lógica Fuzzy.

## 3. Modelar o sistema fuzzy

Apresentar:

- variáveis de entrada e saída,
- conjuntos fuzzy,
- funções de pertinência,
- regras fuzzy utilizadas.

Exemplo:

- Entrada: temperatura
- Saída: velocidade do ventilador

## 4. Implementar em Python (Google Colab)

O notebook deve conter:

- código organizado,
- comentários explicativos,
- gráficos das funções de pertinência,
- execução completa do sistema fuzzy.

Pode ser utilizada biblioteca como:

- `scikit-fuzzy` (`skfuzzy`)

## 5. Apresentar os resultados

Mostrar:

- entradas utilizadas,
- inferência fuzzy,
- resultado final (defuzzificação),
- interpretação do resultado.

***
